Plotly Introduction

What is Plotly? and Why is it useful for Data Visualization?

  • Plotly is a data visualization tool that provides a library for creating interactive and dynamic visualizations in web-based environments.

Plotly is useful for Data Visualization because:

  • Wide variety of Chart types and Customization Options
  • Visually Stunning and Interactive Plots with ease.
  • Real-time collaboration, sharing, and Embedding of visualizations in web pages.
  • Plotly can be used with multiple languages such as Python, R, and Julia

Plotly Express and Plotly Graph Objects

  • Plotly Express: Plotly Express is a high-level and easy-to-use Interface for creating dynamic and quick visualizations.
  • Plotly Graph Objects: Plotly Graph Objects (GO) is a low-level interface for creating more customized and complex visualizations. With Plotly GO, you have full control over creating and customizing your visualizations, including the ability to create subplots, annotations, and custom axes. A Bit more coding is involved with Plotly GO.

However, both Plotly Express and Plotly GO use the same underlying library, so you can use both libraries interchangeably and mix-and-match features from both to suit your needs.

Scatter Plot - Great for visualizing Correlations and Patterns [ Numeric, Numeric ]

fig = px.scatter(df, x='bill_length_mm', y='bill_depth_mm', 
                 labels={"bill_length_mm":"Bill Length", "bill_depth_mm":"Bill depth"},
                title="Scatter Plot",
                color="species",
                symbol="sex",
                hover_data=["year", "island"])
fig.show()

Bubble Chart - Type of Scatter Plot which represents data points as bubbles.

fig = px.scatter(df, x='bill_length_mm', y='bill_depth_mm', 
                 labels={"bill_length_mm":"Bill Length", "bill_depth_mm":"Bill depth"},
                title="Bubble Chart",
                color="species",
                 size="rowid",
                 size_max=20,
                #symbol="sex",
                hover_data=["year", "island"])
fig.show()

3D Scatter Plot - Great for showing the relationship between 3 variables using the x,y, and z-axis.

Bar Chart - Great for Showing Category or Survey or Categorical Count. [ Categorical, Numeric]

fig = px.bar(df, x="species", color="species", animation_frame="year")
fig.show()
